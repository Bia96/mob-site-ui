Grid

The grid is powered by flexbox and requires no container. A column will take up 100% of its containing element, and divide equally and infinitely for each column added. 

<div class="row">
  <div class="column">1</div>
</div>

No Collapse

If the grid shouldn't collapse to 100% width on small screens, add no-collapse to the row.

<div class="row no-collapse">
  <div class="column-90">90%</div>
  <div class="column-10">10%</div>
</div>

Offsets

Add an offset- when you need empty space.

<div class="row">
  <div class="column-25">25%</div>
  <div class="offset-50"></div>

Typography

The font size is set to 1rem on the html element, so all font sizing will cascade from there. Here's a link.

Inline styling remains mostly unchanged: emphasized, italicized, strong, bold, highlighted, underlined, inserted, deleted, strikethrough, and so on.

Headings all have the same line-height and margins, and only the font size is individually defined. A span inside of an h is muted.
