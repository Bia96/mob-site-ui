Mise en Place

(mi zɑ̃ ˈplas)

Mise en place is a French culinary term that means "putting in place", or getting set up. It's a simple starter pack built with Sass that provides helpful styling for buttons, forms, tables, lists, and typography, as well as a grid system for templating.

Containers

There are three container sizes for content that are centered with left and right padding. Content without a container will span the width of the viewport.

<div class="small-container">Small</div>
<div class="medium-container">Medium</div>
<div class="container">Large</div>
<div>None</div>
Small container - 800px
Medium container - 1000px
Container - 1200px

Grid

The grid is powered by flexbox and requires no container. A column will take up 100% of its containing element, and divide equally and infinitely for each column added. Columns have no margins and are padding based. Columns will break to 100% width on mobile collapse, which is set to 600px.

<div class="row">
  <div class="column">1</div>
</div>
<div class="row">
  <div class="column">2</div>
  <div class="column">2</div>
</div>

Blockquote

Blockquote is set to have a left border, with the browser default padding removed. A cite will be right aligned and italicized. 

<blockquote>
	Keep it secret. Keep it safe.
	<cite>Gandalf the Gray</cite>
</blockquote>

Default forms will have full-width, block level labels, inputs, selects, and textareas.

  <input type="text" id="name" placeholder="First Name">
  <label for="choose">Select</label>
  <select id="choose">

    <option value="option-1">Option 1</option>
    <option value="option-2">Option 2</option>
  </select>
  <label for="comments">Additional Comments</label>
  <textarea id="comments"></textarea>
  <label>
    <input type="checkbox">

</form>

Screen Reader Text

Accessibility is important - if you have a label that should be hidden but accessible, or use an icon as a link, use the screen-reader-text class to hide content visually. 

<span class="screen-reader-text">Hidden</span>
