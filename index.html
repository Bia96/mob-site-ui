Mise en Place

(mi zɑ̃ ˈplas)

Mise en place is a French culinary term that means "putting in place", or getting set up. It's a simple starter pack built with Sass that provides helpful, browser-consistent styling for buttons, forms, tables, lists, and typography, as well as a grid system for templating.

Getting Started

First, Mise en Place was created to give consistent styling to HTML elements.

Here is a test page with Mise en Place default CSS applied to common HTML elements.

Additionally, Mise en Place is built with Sass to quickly and easily transform the look and feel of your site, as well as organizing your files into a structured Sass project. View more about that here.

There is a flex-based grid system built in, as well as some useful helper classes.

View the source on GitHub.

Containers

There are three container sizes for content that are centered with left and right padding. Content without a container will span the width of the viewport.

<div class="small-container">Small</div>
<div class="medium-container">Medium</div>
<div class="container">Large</div>
<div>None</div>

Grid

The grid is powered by flexbox and requires no container. A column will take up 100% of its containing element, and divide equally and infinitely for each column added. Columns have no margins and are padding based. Columns will break to 100% width on mobile collapse.

<div class="row">
  <div class="column">1</div>
</div>

Repeating Grid

A grid that repeats and wraps to the next line. A grid-3 class will divide each cell into 1/3, or 33% of the container. Any leftover cells will fill up the rest of the space. Grid cells will break to 100% on mobile collapse.

<div class="grid-3">
	<div class="cell">33%</div>
	<div class="cell">33%</div>
   <!-- etc -->
</div>

Typography

The font size is set to 1rem on the html element, so all font sizing will cascade from the root.

Headings all have the same line-height and margins, and only the font size is individually defined. A span inside of an h is muted.

Blockquote

Blockquote is set to have a left border, with the browser default padding removed. A cite will be right aligned and italicized. 

<blockquote>
	<p>Keep it secret. Keep it safe.</p>
	<cite>Gandalf the Gray</cite>
</blockquote>

Screen Reader Text

Accessibility is important - if you have a label that should be hidden but accessible, or use an icon as a link, or want to skip navigation, use the screen-reader-text class to hide content visually. 

<a class="screen-reader-text" href="#content">Skip Navigation</a>
