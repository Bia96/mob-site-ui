Grid

The grid is powered by flexbox and requires no container. A column will take up 100% of its containing element, and divide equally and infinitely for each column added. Columns have no margins and are padding based. Columns will break to 100% width on mobile collapse.

<div class="row">
  <div class="column">1</div>
</div>
<div class="row">
  <div class="column">2</div>
  <div class="column">2</div>
</div>

Percentage Widths

Percentage based column widths can be added. A class of column-25 creates a 25% wide column. 

<div class="row">
  <div class="column-25">25%</div>
  <div class="column-50">50%</div>
  <div class="column-25">25%</div>
</div>

Fixed Widths

Insert fixed width colums and regular colums will fill in the available space. A class of column-200 creates a 200 pixel wide column. 

<div class="row">
	<div class="column"></div>
	<div class="column"></div>
	<div class="column-200">200px</div>
</div>

Repeating Grid

A grid that repeats and wraps to the next line. A class of grid-4 will divide each cell into 1/4, or 25% of the container. Grid cells will break to 50% width on mobile collapse.

<ul class="grid-4">
	<li></li>
	<li></li>
   <!-- etc -->
</ul>

Typography

The font size is set to 1rem on the html element, so all font sizing will cascade from there. Here's a link.

Inline styling remains mostly unchanged: emphasized, italicized, strong, bold, highlighted, underlined, inserted, deleted, strikethrough, and so on.

Headings all have the same line-height and margins, and only the font size is individually defined. A span inside of an h is muted.

Buttons

Button elements, links with a class of .button, and input elements should all render exactly the same.

<button>Button</button>
<input type="submit" value="Submit">
<input type="button" value="Input">
<a class="button">Link</a>
Link

Accent Buttons

Adding a class of accent will display a secondary color.

<button class="accent">Button</button>
Link

Muted Buttons

Adding a class of muted will create ghost buttons.

<button class="muted">Button</button>
Link

Round Buttons

Adding a class of round will create rounded buttons.

<button class="round">Button</button>
Link

Full Width Buttons

Adding a class of full will create full-width, block level buttons. Full buttons can be combined with other styles.

<button class="full">Button</button>

Forms

Default forms will have full-width, block level labels, inputs, selects, and textareas.

<form>
  <label for="name">Name</label>
  <input type="text" id="name" placeholder="First Name">
  <label for="choose">Select</label>
  <select id="choose">
    <option disabled selected>Please select</option>
    <option value="option-1">Option 1</option>
    <option value="option-2">Option 2</option>
  </select>
  <label for="comments">Additional Comments</label>
  <textarea id="comments"></textarea>
  <input type="submit" value="Submit">
</form>
